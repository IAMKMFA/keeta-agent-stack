# `@keeta-agent-stack/policy`

Policy engine: rule definitions, composition (allOf/anyOf/not), config helpers, and the default
policy pack used by the worker. Pluggable via `defaultPolicyConfigFromEnv()`.

Part of the [Keeta Agent Stack](https://github.com/IAMKMFA/keeta-agent-stack). For end-user agent
code, start with [`@keeta-agent-stack/sdk`](https://www.npmjs.com/package/@keeta-agent-stack/sdk) or
[`@keeta-agent-stack/agent-runtime`](https://www.npmjs.com/package/@keeta-agent-stack/agent-runtime);
this package is a building block.

## Status

`0.1.0-alpha` — API may shift before `0.1.0`.
